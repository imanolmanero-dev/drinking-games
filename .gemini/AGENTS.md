# BeberGames — Instrucciones para Agentes

Lee el archivo `AGENTS.md` en la raíz del proyecto. Contiene TODAS las reglas obligatorias.

Puntos críticos:
1. **Anti-Stuttering (Regla #1):** Nunca generar listas >15 ítems en una sola operación. Verificar visualmente después de cada bloque.
2. **Calidad MDX (Regla #2):** Español castellano, tono divertido, 800-1500 palabras, links internos obligatorios.
3. **Build obligatorio (Regla #5):** Siempre ejecutar `npm run build` antes de cerrar una tarea.
